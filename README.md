## 组件名称：july-tools
- 主要用于提供开发工具类

## 使用方式：
- 项目pom.xml文件引入依赖：

``` 
<dependency>
    <groupId>com.july</groupId>
    <artifactId>tool</artifactId>
    <version>1.0.1</version>
</dependency>
```
